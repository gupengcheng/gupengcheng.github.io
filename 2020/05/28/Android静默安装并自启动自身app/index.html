Android静默安装并自启动自身app

Posted on 2020-05-28

静默安装其实就是在用户无感知，无需操作的情况下完成app的安装，主要分为两种情况的处理，即Root和非Root。分别采用了两种方式去实现，这里模仿的是360手机助手的静默安装和闪装的功能，分别也是对应root手机和非root手机的方案。从安装到自启动分为两个步骤实现。

一. 安装的方案（root和非root手机）

1、root机型

静默安装并自启动这一块，很早做应用商店就接触过。主要分为root过的手机和非root手机的处理。我们都知道root过的手机在应用获取到root权限之后，是可以执行静默安装的命令的，其实就是通过su命令去执行pm install -r安装app。

2、非root机型

而对于非root的普通手机，是无法执行这个命令的，但是360给我们提供了一种参考方案，就是获取android提供的无障碍服务权限，让用户或者如果是ToB的应用，在给客户安装的时候就打开无障碍的权限开关，开启自动安装。可以实现监听系统安装应用，自动帮助用户点击如：“确定”、“安装”、“打开”之类的按钮。
